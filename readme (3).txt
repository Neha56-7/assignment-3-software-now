Spot the Difference

A desktop game built with Python, Tkinter, and OpenCV.

Features:
- Loads an image and displays the original and a modified copy side by side.
- Randomly creates 5 non-overlapping differences using OpenCV.
- Lets the player click the modified image to find the differences.
- Tracks remaining differences, mistakes, and cumulative score.
- Reveals unfound differences on request.

Requirements:
- Python 3
- opencv-python
- pillow
- numpy

Run:
python spot_the_difference.py

Install dependencies:
pip install opencv-python pillow numpy
